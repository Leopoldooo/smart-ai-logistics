# Smart AI Logistics

## Overview

This project predicts risky driver behavior using Machine Learning and automates data processing using FastAPI and n8n.

---

## Features

- Machine Learning prediction
- FastAPI REST API
- SQLite database
- n8n automation
- Streamlit dashboard
- Interactive charts

---

## Technologies

- Python
- FastAPI
- Scikit-learn
- SQLite
- Streamlit
- Plotly
- n8n

## Running the Project

### Install

```bash
pip install -r requirements.txt
```

### Run FastAPI

```bash
python -m uvicorn api.main:app --reload
```

### Run Streamlit

```bash
streamlit run dashboard/app.py
```

### Run n8n

```bash
n8n
```
